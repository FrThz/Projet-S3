# Projet-S3

Nous avons pour objectif de manipuler une liste de données contenants les XP obtenus par 100 élèves dans chaque domaine d'une UE et 
la mention 1 ou 0 en fonction de si l'élève à validé ou non.

Projet finale : réaliser un diagramme en camembert présentant le pourcentage d'élèves qui ont validés ayant une dominance dans tel ou tel 
domaine de l'UE.
